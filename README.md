# Gargot — Tienda Online

Este repositorio contiene el proyecto completo con:
- Código fuente (PHP + HTML + CSS + JS)
- Base de datos exportada (`gargot.sql`)
- Manual de uso completo en PDF

Manual de uso (descargar):
https://raw.githubusercontent.com/jorgepaslaguia/gargot/main/manualgargot.md

## Cómo probarlo
1. Importar `gargot.sql` en tu servidor MySQL
2. Colocar la carpeta del proyecto en tu servidor (localhost)
3. Ajustar credenciales de BD en el archivo de configuración
4. Directorios importantes:
   - `/admin` → Administración
   - `/user` → Zona de usuario
   - `/assets` → CSS/Imágenes
